Background:

Windows 7 is the operating system for the PC’s that we will use in the Software Development Laboratories (Walsh 101 and Walsh 103). 

You will use the Documents folder within your user folder on the lab machines to store all your work. Your user folder is accessible from any machine in this lab or in the Walsh 103 lab.  The organization principle you use to create subdirectories within your Documents folder is a matter of taste.  

It is common practice when programming in Java to create a project for each program such as a lab assignment.  A project is the collection of files used to build the application program. The development environment Eclipse will create a new folder for each project you create this semester. You will create a project named Lab1 for today's lab.

To retrieve a file from a subdirectory, you must know how to navigate the file system (the hierarchy of folders/subdirectories) of the disk.  You will learn how to navigate the file system in the lab exercises.  

Files are named with two parts.  The first part is the name and the second part is called the extension.  The two parts are separated by a period.  The extension identifies the type of file it is.  Usually the extension identifies the application program that uses the file.  For example, you will use a file in the exercises of this lab named Hello.java.  The .java extension indicates that it is a file containing a program written in the Java language. Note: if file name extensions are not visible, you can set the view options on Windows Explorer to display them. We illustrate this below.
