Programming Assignment 1:  “Happy Birthday”

 

One of the most widely sung songs, if not the most popular song, is “Happy Birthday”.  In this first project you will write a program to display the lyrics to this song.

 

Write a Java program that displays the lyrics of the song “Happy Birthday” in a single dialog box.  Format the lyrics as a poem with lines of text.  Include proper punctuation.

 

Notes:

-           Use the DisplayMessage program from Lab 1 as a pattern for this program. 

-           Use JOptionPane for display.  Display all the output in one Message dialog.

-           Document the program with comments as we have illustrated in the programs encountered in the lab assignments.

-           Follow the programming style we have illustrated in terms of indentation, meaningful identifiers, and spacing. 

 

Turn in:

-           Listing of the source code of your program.

-           Listing of the output.

-           E-mail a copy of your source code file to your instructor as an e-mail attachment.
